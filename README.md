# 🚀 Ragu Tools

> **Fast, Private, Offline-First Developer Utilities**

**Ragu Tools** is a lightweight, high-performance suite of client-side web tools designed for software engineers, designers, and digital creators. Built with a strict **privacy-first philosophy**, every operation—from image-to-Base64 encoding to cryptographic password generation—runs entirely inside your local browser memory. No data ever touches a remote server.

---

## ✨ Key Highlights

* **🔒 100% Private:** Operations execute locally via JavaScript inside your browser tab. Zero network traffic, zero analytics tracking, and zero server logging.
* **⚡ Instant Speed:** Zero network latency. Conversions, encodings, and string outputs happen instantaneously.
* **📱 High-Contrast Design:** Clean Dark and Light modes built for high readability across desktop, tablet, and mobile browsers.
* **🌐 Offline-First:** Fully functional without an active internet connection once loaded.

---

## 🛠️ Available Utilities

* **⏰ Unix Timestamp Converter:** Convert Unix timestamps to UTC, Local, and ISO dates or encode human dates to Epoch numbers.
* **🔤 Text Case Converter:** Transform strings between `camelCase`, `PascalCase`, `snake_case`, `UPPERCASE`, and `Title Case`.
* **🔑 Password Generator:** Generate cryptographically secure passwords with custom length and character rules.
* **🔗 URL Encoder & Decoder:** Safely encode special characters for URLs or decode percent-encoded query strings.
* **📦 Base64 Encoder & Decoder:** Convert plain text, files, or clipboard images (`Ctrl+V`) into Base64 strings, Data URIs, or back to raw formats.
