URL Encoder & Decoder

Smart URL encoder. If the input is a valid URL, only query parameter values are encoded. Otherwise, the entire input is encoded.

What is URL encoding?

URL encoding, also called percent-encoding, converts characters that aren't allowed in a URL — spaces, ampersands, slashes, unicode characters, and more — into a %-prefixed hexadecimal format so the URL can be transmitted correctly over the web. For example, a space becomes %20 and an ampersand becomes %26. This matters most inside query string values, where an un-encoded character can break the URL or get parsed as a separate parameter.

How this tool works

Paste any text or full URL into the input box above and click Encode. If the input is recognized as a valid URL, only the values of its query parameters are encoded and the rest of the URL (scheme, host, path) is left readable. If the input isn't a URL, the entire string is percent-encoded instead. Check Encode entire input to force full encoding regardless of URL detection. Use Swap to flip the input and output, which is a quick way to decode a previously encoded string.

Common use cases

  • Preparing query parameter values (search terms, redirect URLs, tokens) for safe use in a link
  • Decoding a percent-encoded URL to inspect its real value during debugging
  • Escaping special characters before building an API request URL
  • Sanitizing user-generated text before appending it to a URL

This tool runs entirely in your browser using the native encodeURIComponent and URL APIs. Nothing you type is sent to a server or stored anywhere.
