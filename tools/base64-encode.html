What is Base64 Encoding?

Base64 encoding converts binary data into a sequence of 64 printable ASCII characters. It is commonly used when transmitting binary data (like images, fonts, or files) over media designed to deal primarily with textual data, such as HTML, CSS, JSON, or email systems.

Key Features

  • Clipboard Image Support: Copy any image (e.g., via screenshot or browser) and press Ctrl + V to instantly encode it.
  • Drag & Drop File Upload: Upload any file or image directly from your local drive to get its Base64 string.
  • UTF-8 Safe Encoding: Supports full unicode encoding for international characters and emojis.
  • 100% Client-Side Privacy: File encoding runs entirely inside your browser. No files are uploaded to any server.

How to Encode Text or Images to Base64

  1. Type text in the input box, drop a file, or paste an image directly using Ctrl + V.
  2. If text was entered, click Encode Text (or press Ctrl + Enter).
  3. If an image was uploaded or pasted, an instant live preview will pop up along with the Base64 Data URI.
  4. Use the copy buttons to get either the raw Base64 string or the formatted HTML/CSS Data URI.
