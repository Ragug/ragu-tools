What is URL decoding?

URL decoding reverses percent-encoding, turning %-prefixed hexadecimal sequences back into their original characters. For example, %20 becomes a space and %26 becomes an ampersand. Browsers, servers, and APIs commonly encode URLs and query parameters this way so that reserved and non-ASCII characters can travel safely inside a URL — decoding lets you read that value back in its original form.

How this tool works

Paste any percent-encoded URL, query string, or text into the box above and click Decode. The tool runs it through the browser's native decoding function and shows the readable result below. If the input contains invalid or malformed encoding, the error is shown in the result box so you can spot the problem instead of getting a silent failure.

Common use cases

  • Reading the real value of an encoded query parameter while debugging a redirect or API call
  • Converting an encoded link from a browser address bar or server log back to plain text
  • Checking exactly what a percent-encoded token, search term, or filename actually contains
  • Verifying that a URL encoder tool produced the expected output, by decoding it back

This tool runs entirely in your browser using the native decodeURIComponent API. Nothing you type is sent to a server or stored anywhere.
