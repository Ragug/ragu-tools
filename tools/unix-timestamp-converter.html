What is a Unix Timestamp?

A Unix timestamp (also known as Epoch time or POSIX time) measures time as the total number of seconds elapsed since 00:00:00 UTC on January 1, 1970 (excluding leap seconds). It is the universal standard for logging events, storing datetime fields in databases, and syncing systems globally.

Seconds vs. Milliseconds

  • Seconds (10 Digits): Standard Unix epoch timestamp used across C, C++, Python, PHP, and SQL databases.
  • Milliseconds (13 Digits): Used natively in JavaScript (Date.now()) and Java for sub-second precision.

Developer Quick Reference

  • JavaScript: Math.floor(Date.now() / 1000)
  • Python: import time; int(time.time())
  • SQL (MySQL): SELECT UNIX_TIMESTAMP();
  • Bash / Linux: date +%s
